implementation

GitHub Pages erlaubt es nicht, kein Index.html file zu haben. Aber damit fuer jedes Lied ein Separates Artikel erstellt werden kann, und da ein QR-code pro Lied existiert, welcher auf eine separate Seite zeigen muss, muss also pro Artikel eine separate Seite erstellt werden.
Dies bedeutet das pro Seite ein html-file existiert.
Es gibt auch weitere limitationen, wie zum Beispiel das kein Server-side rendering verwendet werden darf. Dies bedeutet das die Ganze Website nur aus html, css und einfachem Javascript bestehen darf. Also Javascript-Frameworks oder andere tools wie zum Beispiel die Google Material Web Library konnten nicht benutzt werden.

Wer hat diese Website Erstellt?

Die einzelnen Artikel wurden wie Oben beschrieben von Schülern aus der Klasse 2AM geschrieben. Die Design-Idee des Menu buttons, welcher animiert wird, stammt von Luka Adjancic. Alles andere, und dies beinhaltet die Implementation des Menu-Buttons, des Popup-Menus in Javascript, die Konversion aller Artikel zu html dateien und dieses Artikel wurden von Nikolay Voropayev erstellt.
